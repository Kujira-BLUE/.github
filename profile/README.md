## Practical Fixes and Safe Habits

Staking action pending: check gas settings, validator selection, and whether the wallet has enough available KUJI outside the bonded amount. Bridge delay: review Kujira BLUE IBC bridge status, confirm source chain completion, and avoid repeating transfers until the route state is clear.

Swap unavailable: confirm the asset pair is supported in Kujira BLUE swap and that wallet balances are sufficient for fees. Governance vote missing: make sure the proposal is active, the wallet is connected, and staking status allows Kujira BLUE governance voting. Developer action unclear: use Kujira BLUE developer tools carefully, review contract details, and test small interactions before executing larger changes.
